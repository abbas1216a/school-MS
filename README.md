# School Management System (SMS)

A simple School Management System built using HTML, CSS, and JavaScript. This project provides a user-friendly interface for managing school-related activities through a web dashboard.

## Project Structure

```
sms/
│
├── index.html
├── dashboard.html
├── register.html
│
├── css/
│   ├── style.css
│   └── dashboard.css
│
├── js/
│   ├── auth.js
│   └── dashboard.js
│
└── .nojekyll
```

## Features

* User Registration
* Login Authentication
* Dashboard Interface
* Responsive Design
* HTML, CSS, and JavaScript Based

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your web browser.
3. Register a new user.
4. Access the dashboard after authentication.

## GitHub Pages Deployment

After uploading the project to GitHub:

1. Go to **Repository Settings**.
2. Open **Pages**.
3. Under **Source**, select:

   * Branch: `main`
   * Folder: `/ (root)`
4. Click **Save**.

Your website URL will be:

```
https://abbas1216a.github.io/school-MS/
```
